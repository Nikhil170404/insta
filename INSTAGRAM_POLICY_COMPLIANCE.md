# Instagram Platform Policy Compliance

## Spam Prevention
✓ Rate limiting: Max 100 DMs/hour (well below Instagram's limits)
✓ One DM per user per keyword (prevents spam)
✓ User must comment first (no unsolicited messages)
✓ Clear opt-out: Users can block/report as spam

## User Privacy
✓ Data encryption at rest (AES-256)
✓ HTTPS for all transmission
✓ No data selling/sharing
✓ GDPR-compliant deletion within 30 days
✓ Verified Domain: replykaro.com

## Authentic Communication
✓ Messages come from verified Business/Creator accounts
✓ Clear sender identity (account owner)
✓ "Human Agent" tag (shows it's automated)
✓ No impersonation

## Prohibited Content
✓ No support for: Adult content, hate speech, violence, illegal goods
✓ Keyword filtering for offensive terms
✓ Manual review option for flagged content

## Data Use
✓ Only access data necessary for service
✓ Comments + Messages only (no followers list, no insights)
✓ Tokens stored securely, refreshed every 60 days
